## thinkt stats

Show usage statistics from the index

### Synopsis

Display aggregate statistics from the SQLite session index.

Shows project, session, entry, and token counts along with top tool usage.
Run 'thinkt sync' first to populate the index.

```
thinkt stats [flags]
```

### Options

      --json   Output as JSON
