## thinkt docs markdown

Generate markdown documentation

### Synopsis

Generate markdown documentation for all thinkt commands.

By default, generates plain markdown suitable for GitHub wikis and basic
documentation sites. Use --hugo to generate markdown with YAML front matter
for Hugo static site generator.

```
thinkt docs markdown [flags]
```

### Options

```
  -h, --help   help for markdown
      --hugo   generate Hugo-compatible markdown with YAML front matter
```
