## thinkt web lite

Open the lightweight web interface

### Synopsis

Open the thinkt lite web interface (debugging and development view).

```
thinkt web lite [flags]
```

### Options

```
  -h, --help          help for lite
      --host string   server host (default "localhost")
      --no-open       don't auto-open browser
  -p, --port int      server port (default 8784)
```
