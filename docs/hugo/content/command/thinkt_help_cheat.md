## thinkt help cheat

Quick-reference command tree

### Synopsis

Print a compact cheat sheet showing every command and subcommand.

Depth flags control how many levels to show (default: 3).

```
thinkt help cheat [flags]
```

### Options

```
  -h, --help    help for cheat
      --json    output as JSON
      --md      output as Markdown
  -1, --one     show 1 level (top-level commands only)
  -3, --three   show 3 levels (default)
  -2, --two     show 2 levels
```

```

```
